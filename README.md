WP-Bootstrap-2.1
================

A very basic wordpress theme based on Twitter Bootstrap 2.1.

Features
--------

* Full Width Template
* Left Side Bar Template
* Responsive

Installation
------------

Navigate to wordpress theme directory

    git clone git@github.com:Alan01252/WP-Bootstrap-2.1.git
  
Activate theme
